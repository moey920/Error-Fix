## 구구단을 제대로 외자

2단부터 5단까지 출력하는 코드를 짰는데, 원하는 대로 동작하지 않아요.
코드를 잘 읽고 틀린 부분을 모두 고쳐서, 아래의 결과를 출력하게 해 주세요!

```
for left in range(2, 6):
    print(str(left) + "단")
    for right in range(1, 10):
        # "2단", "3단"을 출력합니다.
        product = left * right
        line = str(left) + " x " + str(right) + " = " + str(product)
        print(line)
```
